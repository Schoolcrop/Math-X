(Q39)Find the relation between x and y such that the point (x,y) is equidistant form the points (-2,8) and (-3,5).

Answer

Let A(-2,8),B(-3,-5) and P(x,y). If P is equidistant from A,B and then PA =PB

Distance formula=√((x2 - x1)2 + (y2 - y1)2)

PA=√((+2)2 + (-8)2)

=√(x2 + 4x + + y2 - 16y + )

=√(x2 + y2 + 4x - 16y + )

PB=√((+3)2 + (+5)2)

=√(x2 + 6x + + y2 + 10y + )

=√(x2 + y2 + 6x + 10y + )

Now PA=PB

=√(x2 + y2 + 4x - 16y + ) = √(x2 + y2 + 6x + 10y + )

Squaring on both sides we get

=x2+y2+4x-16y+=x2+y2+6x+10y+
