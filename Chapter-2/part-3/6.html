(Q6).Consider the sets ∅ , A = {1, 3}, B = {1, 5, 9}, C = {1, 3, 5, 7, 9}. Insert the symbol ⊂ or ⊄ between each of the following pair of sets.

(i) ∅ ….. B

∅ ⊂ B, as ∅ is a subset of every set.
∅ ⊄ B, as ∅ is a subset of every set.
∅ = B, as ∅ is a subset of every set.
∅ ≠ B, as ∅ is a subset of every set.

(ii) A ….. B

A ⊂ B, as 3 ∈ A but 3 ∉ B.
A ⊄ B, as 3 ∈ A but 3 ∉ B.
A = B, as 3 ∈ A but 3 ∉ B.
A ≠ B, as 3 ∈ A but 3 ∉ B.

(iii) A ….. C

A = C as 1, 3 ∈ A also belong to C.
A ⊄ C as 1, 3 ∈ A also belong to C.
A ⊂ C as 1, 3 ∈ A also belong to C.
A ≠ C as 1, 3 ∈ A also belong to C.

(iv) B ….. C

B ⊄ C as every element of B is also an element of C
B ≠ C as every element of B is also an element of C
B = C as every element of B is also an element of C
B ⊂ C as every element of B is also an element of C
